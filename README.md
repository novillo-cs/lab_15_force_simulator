## Directions:

Extend the force simulator we created in claas with a feature of your own design. This is your chance to demonstrate your creativity using objects, PVectors, forces, user input, or collision logic.

Requirements:

- Your feature must meaningfully extend the existing simulator, not just change colors.

- You must add at least one new method to the Ball class, or create a new class that interacts with Ball.

- Your code must run without errors.

- Include a file (txt, md, docx, pdf) with a brief description explaining what you added and how to use it (which keys/mouse buttons do what).

Here are some ideas:

- Obstacles: static rectangles the balls bounce off, placed by the user.
  
- Bounciness decay: each bounce reduces velocity by 10%, so balls eventually settle.
  
- Floor friction: when a ball is touching the bottom, reduce its horizontal velocity each frame.

- Gravity: press up/down arrow keys to increase or decrease gravity strength; display the current value on screen.
